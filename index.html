UofT Class Distances (2014)
This is a web service I created for UofT students to check how long it would take to walk between adjacent classes in their school timetables.
By utilizing the Griddy (a course planning tool for UofT students) timetable sharing feature, I was able to find all the courses a student is taking and find the locations of the lectures and tutorials. The Google Maps API is used to find the exact distance and duration of each walk.

Q-2048 (2014)
As a culminating project for my grade 12 high school computer science class, I decided to make a web-based multiplayer version of the popular 2048 game originally developped by Gabriele Cirulli.
Using a different iterative approach than the original game's method, I developed an algorithm for merging similar tiles according to the game's description. By utilizing web sockets, I developed a system for real-time communication between multiple 2-client game sessions.

FIRST Robotics Robot (2014)
FIRST is a non-profit community that organises international robotics competitions for high school students. I took on the challenge of building the software for my school's robot participating in the 2014 FIRST Robotics Competition (FRC).
Our robot was designed to maneuver a field attempting to pick up excercise balls that were 2-feet in diameter. The objective was then to throw these excercise balls into the opponents goals to score points. Our robot collaborated with 2 others in order to form an alliance to defeat other teams.

YouTube Chat Rooms (2013)
A project inspired by a service I attempted to create at UofTHacks 2013, this was a web application I developed to allow for Google-authenticated users to watch YouTube videos in sync with other users in a managed chat room.
The creator of each room is the administrator which includes the responsibilities of playing, pausing, and skipping parts in the certain video. Any change the administrator makes to the video reflects, in real-time, to the other users who happen to be in the group. Users are also able to chat as well.
